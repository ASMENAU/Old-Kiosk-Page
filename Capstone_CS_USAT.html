User System of Astrogeology Technologies
Team Members: Kyle McGinn, Megan Backus, Mikal Ustad & Zack Ellett
Abstract:
The Astrogeology Science Center at the United States Geological Survey(USGS) uses a suite of software called ISIS for processing images received from NASA planetary missions. The 300 programs that makeup ISIS are combined in different ways to create workflows which assist scientists in image analysis. Currently, these workflows are computer scripts written by hand and require an advanced knowledge of the ISIS system. The USAT project is a unified interface that addresses the shortcomings of the current workflow process, while maintaining compatibility with the existing ISIS system. A Graphical User Interface increases the usability of the system. USAT's web based design alleviates current cross-platform limitations by making the application accessible from any web browser.
Background:
Isis is an image processing software package. The focus of the software is to manipulate imagery collected by current and past NASA planetary missions sent to Mars, Jupiter, Saturn, and other solar system bodies. If you're familiar with commercial image processing packages (for example: Adobe Photoshop, Envi, or ERDAS Imagine) you'll recognize many of the standard image processing operations Isis is capable of, such as contrast stretch, image algebra, filters, and statistical analysis.
