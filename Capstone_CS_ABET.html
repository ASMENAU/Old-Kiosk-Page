WEB 2.0 Accreditation and Departmental Management System
Team Members: Michael Brooks, Owain Moss, Jonah Hirsch, Eddie Hillenbrand & Kyoko Makino
Abstract:
NAU's Computer Science program first received the Accreditation Board for Engineering and Technology's (ABET) accreditation in October 1996, and has maintained the status since. To receive ABET accreditation, an academic program must define program objectives and outcomes that meet ABET guidelines and develop a continuous monitoring process. Collected data must then be analyzed continuously over the seven year timespan between official reviews. Given the complexity and diversity of the data collection process and schedule, it's easy for data collection or analysis tasks to be forgotten, overlooked, or neglected. The objective of Team Zabeta's Capstone project is to create a system which provides a web interface that supports the ABET data collection and management.
Background:
In order for an institution to receive ABET accreditation, the institution must create courses and analyze the instruction results to make sure that they meet ABET guidelines, then go through a long process of documenting all of their activities. The problem with this deceptively easy model hides in the process of documentation and archiving them.
