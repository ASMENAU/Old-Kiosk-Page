Urban Concept Vehicle (Suspension and Chassis Team)
Team Members: Mohammed Ramadan, Noe Caro, Brian Martinez, Chase Beatty
Abstract:
For our senior mechanical engineering capstone design class we are a participating group in the Urban Concept Vehicle project. Our duty for this project is to create a rigid frame as well as functional suspension system for a car which would be used in an urban setting.
Design Objective:
Our team, Team SPAT, will be making a truss type chassis frame with thin wall aluminum tubing. For the front suspension we will be using an A-arm style suspension. While for the back we will be using a trailing arm/linkage style suspension.
