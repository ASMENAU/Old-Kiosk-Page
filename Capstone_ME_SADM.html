Simulated Aneurysm Deployment Model
Team Members: Duane Bitsilly, Juan Fuentes, Jonathan Silva, Jennifer Torres, Shadi Bafaraj, Thermaine Johnson, Tom Ribaudo, Drey Vaughn & Justin Wong
Abstract:
The goal of this project is to create a working model to simulate an abdominal aortic aneurysm and demonstrate how the Gore® Excluder® AAA Endoprosthesis (Excluder) treats this life threatening issue.
Design Objective:
An abdominal aortic aneurysm is a buildup of plaque in a human that reduces blood flow in an aorta. This plaque causes the aorta to expand in order for the blood to still pump through the body. If the aorta continues to expand, it could burst, and this person would start tobleed internally . Gore has created a product that can help to save a humans life from this tragic event. The design will consist of a temperature sensor, pressure sensor, flow meter, data acquisition system (DAQ) that can display a graphical user interface (GUI), a field programmable gate array (FPGA), and a physical model that simulates basic human properties.
